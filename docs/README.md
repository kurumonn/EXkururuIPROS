# docs 公開ポリシー

このディレクトリは公開配布用の文書と内部検証成果物が混在します。
公開する前に、以下の区分を確認してください。

## 公開してよいもの

- `PUBLIC_TEST_DATASET.md`
- `sample_threat_intel_feed.json`
- `README.4stack.md`
- 公開 README が参照する契約文書や利用説明

## 内部扱い

- `*_latest.json` の品質ゲートや性能計測結果
- 内部閾値、チューニング値、評価ノート
- 実コーパスの場所や実案件の運用導線

内部扱いの成果物は、配布版では `docs/` から外すか、別の private 保管先へ移してください。
